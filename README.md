## Installation Guide

To get started with the app, follow these simple steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/imdb-clone.git

   ```

2. Install the required dependencies:

   ```bash
   npm install

   ```

3. Launch the app:

   ```bash
   npm run android



   ```
